Optional Items:
---------------

1. PPMixed.ipl
Contains extra props for SAxVCxLC Mixed, https://www.mixmods.com.br/2021/10/gta-mixed-lcvcsa.html

2. Procobj.dat & surfinfo.dat
Contains some more procedural generated objects. Small paper, rubbishes etc on streets & pavements.
Put them inside the modloader/Project Props folder to load them.

Note: They are a bit buggy. Sometimes props can appear on walls!

3. lae_stream7.ipl ( aka weeds.ipl )
Contains a lot of weed placements for LS. This is really heavy and may decrease a LOT of fps.
Put this inside the ipl_src folder and run Install ProjectProps.bat
